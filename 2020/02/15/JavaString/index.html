JavaString

Java

String对象

1.概述

public final class String
extends Object
implements Serializable, Comparable, CharSequence

  1. 一旦被初始化就不能改变
  2. String s1 = “abc”;和String s2 = new String(“abc”);区别
    • 前者只创建一个对象,后者创建两个对象

2.常用方法

  1. 获取

    1.1 字符串长度

    • int length();

    1.2 根据位置获取字符

    • char charAt(int index);

    1.3 根据字符/字符串获取第一个位置

    •                       int indexOf(int ch);
                       int indexOf(int ch,int fromIndex);
                        int indexOf(String str);
                        int indexOf(String str,int fromIndex);
                        int lastIndexOf(int ch);
                       int lastIndexOf(int ch,int fromIndex);

3.2 将基本数据类型和char[]转换成字符串

  • static String valueOf(dateType data);

3.3 将字符串转换为字符数组

  • char[] toCharArray();

3.4 字节数组转换为字符串

  • String(byte[] bytes);
  • String(byte[] bytes, int offset, int length);

3.5 字符串转换成字节数组

  • byte[] getBytes();

3.6 把字符串转换成大写/小写

  • String toLowerCase();
  • String toUpperCase();
  • 特殊之处:字符串和字节数组转换过程中,可以指定编码表,所以在处理编码问题时可以以此来进行转换。

  1. 替换

    • String replace(char oldChar,char newChar);
    • String replace(CharSequence target,CharSequence replacement);

  2. 切割

    • String[] split(String regex);

  3. 子串

    • String substring(int beginIndex);
    • String substring(int beginIndex,int endIndex);

  4. 去除空格,比较

    • String trim();–忽略前导空白和尾部空白

    • int compareTo(String anotherString);

3.StringBuffer(缓冲区容器对象):jdk1.0

  1. 简介

    • public final class StringBuffer

      • extends Object
        • implements Serializable, CharSequence

    • 线程安全的可变字符序列。

    • 本身是一个容器，创建以后始终只有一个容器对象,本身并不会新建容器对象。

  2. 常用构造方法

    • StringBuffer()
      构造一个其中不带字符的字符串缓冲区，其初始容量为 16 个字符。
    • StringBuffer(String str);

3.常用方法

  1. 存储

    • StringBuffer append(dataType data);
    • StringBuffer insert(int offset,dataType data);

  2. 删除

    • StringBuffer delete(int start,int end);
    • StringBuffer deleteCharAt(int index);

  3. 获取

    • char charAt(int index);

  1. 修改

    • void setCharAt(int index,char ch);
    • StringBuffer replace(int start,int end, String str);

  2. 反转

    • StringBuffer reverse();

4.StringBuilder jdk1.5

  1. 简介

    • public final class StringBuilder

      • extends Object
        • implements Serializable, CharSequence

    • 一个可变的字符序列。该类被设计用作 StringBuffer 的一个简易替换,不保证同步。单线程时效率高。

5. Java在升级时的三个要素

  1. 提高效率(StringBuilder)
  2. 简化书写(泛型)
  3. 提高安全性
